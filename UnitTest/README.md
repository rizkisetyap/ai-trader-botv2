# UnitTest

This folder contains test scaffolding for both parts of the repo:

- `backend-net.Tests` for the .NET 8 Web API
- `python-worker-tests` for the Python worker side

## Run .NET tests

From `UnitTest/backend-net.Tests`:

```bash
dotnet test
```

## Run Python tests

From `UnitTest/python-worker-tests`:

```bash
pytest
```
